# pricepulse

> competitor pricing pages on a watchlist. weekly / daily / hourly checks. email when something changes.

A browser extension for Chrome, Firefox, and Edge. Manifest V3.

Live: [extensions.voiddo.com/pricepulse/](https://extensions.voiddo.com/pricepulse/)

## What it does

Add a pricing page to your watchlist with one click. pricepulse fetches it on schedule (weekly / daily / hourly) and emails you when:

- a price changes
- a tier name changes
- a feature is added or removed
- the page returns 4xx/5xx (page taken down or restructured)

Free tier: 5 watched pages, weekly checks. Pro: 50 pages, daily, with hourly bursts.
